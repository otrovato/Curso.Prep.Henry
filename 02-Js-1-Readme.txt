Instrucciones
En un archivo de texto separado que debes crear, escribe explicaciones de los siguientes conceptos como si se lo estuvieras explicando a un niño de 12 años. Hacer esto te ayudará a descubrir rápidamente cualquier agujero en tu comprensión.
Variables
Strings
Funciones (argumentos, return)
Declaraciones if
Valores booleanos (true, false)


Variables: las variable es un valor que se guarda para que se pueda usar en el momento que sea necesario. Se declara con la palabra var nombreVariable.
O sea se declara con var seguido del nombre de la variable que le quieras poner, el nombre de la variable siempre tiene que tener una relacion con el valor que va a contener la variable.

String: los String es un valor que almacena texto o cadena de texto. Siempre va entre comillas simples o dobles. Ej: var perro="kimbo"

Funcion: una funcion es una accion que hace algo o no hace nada puede o no devolver un valor. Una funcion se declara funcion nombreFuncion(){
                acccion;}
If: El If es un Si. Es decir, si se cumple la condicion hace una accion o no hace nada.
se declara if (condicion){ hace algo}

Valores Booleanos: Son valores que puede ser Verdades True o Falsos False. Estos valores se utilizan para saber si una condicion es verdadera o falsa.
